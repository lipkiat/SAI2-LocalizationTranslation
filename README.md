# SAI2 Localization Translation Project
# SAI2 本地化翻译项目
Unofficial localization translation
非官方的本地化翻译

# Contribute or add your language
1. `Fork` this project.
2. Copy the `ja` folder and rename it to your language code.
3. Submit a `Pull Request` once completed.

# 贡献或添加你的语言
1. `Fork` 本项目。
2. 复制 `ja` 文件夹并重命名为您的语言代码。
3. 翻译完成后提交`Pull Request`。
